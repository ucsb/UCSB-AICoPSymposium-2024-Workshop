INSTALLATION
------------

 * Install the YouTube Video Gallery module as you would normally install a
   Drupal module.


CONFIGURATION
-------------

YouTube Video Gallery is a simple module to enhance the
SIS&T subtheme paragraph components

Steps to activate:
1- Activate module.
2- Go to the Blades paragraph, /admin/structure/paragraphs_type/blade/fields
3- In field Columns, in section "Paragraph types" select "YouTube Video Gallery".
4- Now, when you add a new blade you can create a youtube playlist column.
